# Blackswan‑style One‑Page (Vite + React + Tailwind)

A minimal one‑page site with a BlackswanQuants‑like aesthetic:
- pitch black background
- violet accent
- outline hero word
- glassy cards
- anchored navbar

## Run
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
npm run preview
```
